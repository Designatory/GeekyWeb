HTML Tags

HTML Tags‚Äã

HTML Tags play an important role in web development, tags hold content to display there are specific tags for specific texts, like if you want to add paragraph then use <p> </p> tags for heading <h1> </h1> and many more...

Closing Tags‚Äã

Closing tags are those that have both opening and closing tags. As previously discussed, if you need to add a paragraph, you will add <p> tag as the opening tag and </p> tag as the closing tag.

<p>This is my paragraph</p>

Opening Tags‚Äã

Opening tags are those tags which have only opening tags, no closing tag. Actually, they also have a closing tag but we don't need to type it, like if we want to add a line break to our webpage we simply add <br> but we will not add </br> because these closing tags don't have any use, these kinds of tags are self-closing tags as we can also just type them as <br /> which indicate both opening and closing tag as well.

This is my text and now i have to break a line and continue to next line. <br />
<!-- or <br/> -->
Now this is my new line
