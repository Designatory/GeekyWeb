HTML Attributes

Introduction‚Äã

They are a type of Html Elements Modifier. Example: HTML attributes can be used to change the shape, size and functionality of HTML Properties

The href Attribute‚Äã

The <a> tag defines the link href attribute specifies a page url. Example:

<a href="https://www.geekyweb.com">GeekyWeb</a>

The Src Attribute‚Äã

the <img> tag is used to add an image to an HTML document. The src attribute specifies the path of the image that is to be displayed. Example :

<img src="demoimage.png" />

The Width and Height Attributes‚Äã

The width and height of the attribute specify the Width and Length of the Image Reflected on the <img> tag (in pixels) Example:

<img src="demoimage.png" width="250px" height="250px" />

The title Attributes‚Äã

The title attribute describes more information about the element. The attribute value of the title is displayed as a tool tip when the mouse cursor is scratched on it. For example

<p title="I am a demo text">Demo Text</p>

The Lang Attributes‚Äã

The lang attributes is included inside the <html> tag which describes the language of the web page

<!DOCTYPE html>
<html lang="en">
  <body></body>
</html>

The Style Attributes‚Äã

The style attribute is used to enhance the style of the element. Example :

<p style="font-size:30px">This is dummy paragraph with font size of 30px</p>

The Alt Attribute‚Äã

The alt attributes specifies the Alternate text for the image. Example:

<img src="logo.jpg" alt="Logo Here" />
